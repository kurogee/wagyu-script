Wagyu-script
---
Wagyu-scriptは、@kurogeが開発した半角スペース区切りが特徴のシンプルなプログラミング言語です。/ Wagyu-script is a simple programming language developed by @kuroge, which is characterized by half-width space separation.

Examples
---

### Hello, World!
```
println "Hello, World!";
```

### FizzBuzz
```
each #from(1 100) > i {
    if ##(i % 15 == 0) {
        println "FizzBuzz";
    } elif ##(i % 5 == 0) {
        println "Buzz";
    } elif ##(i % 3 == 0) {
        println "Fizz";
    } else {
        println i;
    };
};
```
